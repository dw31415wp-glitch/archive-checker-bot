Why this page exists

Some archive and mirror services can trigger malware warnings, browser blocks, or security reviews. This interstitial gives users context before leaving the current page.

The design intentionally uses an article-like layout inspired by Wikipedia: a centered reading column, restrained borders, serif body text, and utility-style actions.

How the link format works

Pass a destination URL in the url query parameter. Only HTTP and HTTPS destinations are accepted.

Example: ?url=https://archive.today/2025.01.01-120000/https://example.org/article

Static HTML, CSS, and JavaScript only. Suitable for direct hosting with GitHub Pages.
